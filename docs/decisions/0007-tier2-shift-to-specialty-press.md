# 0007. 第 2 層を大手紙の教育面から教育専門紙へ転換

- 状態: 採用(0003 の第 2 層を上書き)
- 日付: 2026-04-25
- 関連 PR: TBD(`feat/sprint2-batch2-major-media`)

## 背景

0003 で 3 層ソース構成を定めた際、第 2 層(主要教育メディア)を「朝日 EduA / 毎日 教育 / 読売 こどもと教育 / 共同通信 教育」の 4 紙とした。Sprint 2 Batch 2 でこれら 4 紙の HTML スクレイピング実装に着手する直前、もう一度 RSS 存在 + robots.txt + 利用規約の再確認をサブエージェントで行った結果、想定していなかった事実が判明した。

### 大手紙の現状(2026-04-25 時点)

| 媒体 | RSS | robots.txt | 利用規約 |
|---|---|---|---|
| 読売 こどもと教育 | 2014 廃止 | wget / curl / Scrapy / Nutch を個別 Disallow | 「クローリング、スクレイピング等の自動化された手段」「生成 AI 学習」を明示禁止 |
| 朝日 EduA | なし | 全体許可 | 「ロボット、スパイダー、スクレーパー等」「AI 開発・学習・利用目的のデータ収集」を明示禁止。Perplexity を 2025-08 提訴。EduA 紙媒体は 2024-08 で休刊済 |
| 毎日 教育面 | 教育専用なし(一般速報 RSS のみ) | 通常許可 | 「ロボット、スパイダー、スクレーパー等」を明示禁止。生成 AI 学習も禁止 |
| 共同通信 | 教育/文化カテゴリ RSS あり(`/culture/feed/`、教育記事比率 30%) | 通常許可 | 「複写、複製、翻訳、翻案、改変、頒布」禁止、「引用」のみ許可 |

3 紙(読売 / 朝日 / 毎日)が利用規約で機械収集を明示禁止しており、HTML スクレイピングは規約違反となる。0003 の前提が崩れた。

### 視点の組み替え

PRD §6 の読者像「教員・保護者」に立ち戻ると、本来最も価値が高いのは:

- 教員向け: 教育専門紙(教員向けに書かれており、現場知見が深い)
- 保護者向け: 教育情報サイト(受験・学習・GIGA など保護者の関心領域に特化)
- 速報性: 共同通信 RSS(全体フィードを教育キーワードでフィルタ)

大手紙の教育面は「不特定多数向け」であり、edu-watch のターゲット読者にとっては必ずしも最適ではない。むしろ教育専門紙のほうが本来の読者ニーズに合致する。

## 検討した選択肢

- **A) 共同通信 RSS のみで縮小して進める**(MVP 最小)
- **B) 読売 / 朝日 / 毎日に対して個別に正式許諾申請を試みる**(数週間〜数ヶ月の遅延リスク)
- **C) 「タイトル + URL のリンク集」レベルへ仕様縮小して 4 紙とも残す**(著作権法 32 条「引用」根拠だが、ToS で機械的アクセス自体を禁止している媒体とは抵触)
- **D) 第 2 層を教育専門紙へ総入れ替え + 共同通信は RSS で残す**

## 決定

**D) 教育専門紙への総入れ替え** を採用。

### 第 2 層(改訂後) — 教育専門紙 + 共同通信 RSS

| 媒体 | URL | 取得方式 | 採用区分 |
|---|---|---|---|
| リセマム | `https://resemom.jp/rss20/index.rdf` | RSS(全体)+ ホワイトリスト/NG ワードフィルタ | 即時(Batch 2 v2-α) |
| 日本教育新聞 | `https://www.kyoiku-press.com/rss` | RSS(全体) | Batch 2 v2-β |
| 教育家庭新聞 | `https://www.kknews.co.jp/feed` | RSS(全体) | Batch 2 v2-β |
| 共同通信 | `https://www.kyodo.co.jp/culture/feed/` | RSS(教育/文化カテゴリ) | Batch 2 v2-β |

### 除外する媒体

- **教育新聞(`kyobun.co.jp`)**: RSS なし、robots.txt で AI 系クローラー 60+ を明示 Disallow、ToS で「キュレーション・クリッピングサービス禁止」「営利目的の利用またはその準備行為を禁止」。媒体側のポリシーを尊重し、edu-watch からは取得しない
- **読売 / 朝日 EduA / 毎日**: ToS で機械収集を明示禁止。HTML スクレイピングを実施しない。リンク掲載が必要な記事は編集者が手動で個別リンクとして追加する余地のみ残す

### 引用範囲遵守の要件

第 2 層全媒体について、ADR 0008(引用範囲遵守と削除依頼窓口)で定める 5 要件(出典明示、公式 URL 直リンク、要約は RSS 配信のものをそのまま、タイトル改変禁止、削除依頼窓口の明示)を必須とする。

### コレクションポリシーの公開

`/about` ページに「収集ポリシー」セクションを追加し、(a) 採用ソース一覧、(b) 取得方式(RSS / 引用範囲)、(c) 削除依頼窓口メールアドレス、(d) 出典に関する立場、を明文化する。

## 帰結

### 良い帰結

- 法的リスクが大幅に低減(ToS 違反ゼロ、引用範囲内の取得に統一)
- 読者適合度が向上(教員 → 日本教育新聞、保護者 → リセマム、両者 → 共同通信)
- edu-evidence 本家との差別化が明確化(現場の今 × 専門メディアという軸)
- RSS 採用により実装コストが HTML スクレイピングより低い
- サイト稼働後の保守負荷も低い(媒体側の HTML 変更に追従しなくて済む)

### トレードオフ
- 一般読者にとって馴染みのある「朝日」「毎日」「読売」というブランドが第 2 層から消える
- 教育新聞(媒体としての信頼性は高い)が除外対象となるため、教員向けカバレッジが日本教育新聞 + 教育家庭新聞のみとなる
- リセマムは保護者向け色が強く、受験産業寄りの記事は NG ワードフィルタで除外する運用負荷が発生する
- 共同通信の `/culture/feed/` は教育記事比率約 30% のため、フィルタが必要

## 撤回 / 再検討の条件

- 大手紙各社が RSS 配信を再開し、利用規約上も RSS 経由の引用範囲利用を許諾した場合
- 教育新聞が AI 系クローラー以外の通常クローラーへの方針を緩和した場合(別途許諾申請の可能性も含む)
- リセマムから運用方針について問い合わせがあった場合(ホワイトリストの粒度変更を検討)
- 共同通信の `/culture/feed/` の教育記事比率が大きく低下した場合(別カテゴリフィードへの切替を検討)
- ReseEd(リセマムの教員向け姉妹サイト、`reseed.resemom.jp`)の RSS / ToS 確認が完了し、追加採用が妥当と判断された場合

# 0039. kkn / resemom のノイズフィルター強化

## 背景

ADR 0018 で対象読者を「学校教員・教育関係者」に統一し、ADR 0019 / 0020 / 0024 で resemom の NG ワードを継続的に育成してきた。一方、kkn(教育家庭新聞)は ADR 0007 採択時点から「GIGA 関連の網羅性を担保するため全件採用」というポリシーで運用してきた。

2026-05-11 のセッションで、ユーザーから両 source の運用品質に関する具体的なフィードバックを受領した:

1. 「kkn は不必要な内容が多い」(コンテスト告知・塾/財団主催イベント・食品プロモーションなど)
2. 「resemom も同様、『沖縄、早くも梅雨入り』のような気象記事は不要」

## 検討した選択肢

### A. 中間強度フィルター + ADR 化(採用)

- kkn: `isExcludedByTitle()` 関数を新設、7 カテゴリ NG パターン + 厳格 POSITIVE_OVERRIDE(`教員向け` / `教職員向け` / `学校教員` のみ)
- resemom: 既存 `NG_PATTERNS` に 8 カテゴリ追加(数字統計コピー記事は除外せず)
- 両 source の判断追跡性を本 ADR(統合文書)で保持
- テストスクリプト(kkn 新規 / resemom 拡張)で回帰防止

#### kkn の POSITIVE_OVERRIDE を「厳格」にする理由

ユーザー方針(2026-05-11 セッション)で、POSITIVE_OVERRIDE は「教員向け」「教職員向け」「学校教員」のみに限定する。「中高教員向け」「教員・教委向け」「教育関係者向け」などの複合表記は意図的に override 対象から外し、過剰除外側でスタートする。

理由:
- 「教員 + 〇〇向け」のように対象が拡張された複合表記は、内容が必ずしも教員業務に直結しない(参加対象の幅広さを示す宣伝文句として使われることが多い)
- 過剰通過(本来除外したい記事が混入)よりも過剰除外(本来採用したい記事が漏れる)の方が、運用観測で気づきやすい
- 取りこぼしは後続 ADR で個別救済する余地を残せる

#### 数字統計コピー記事を残す理由

「子供の数は過去最少1,329万人、45年連続で減少…総務省」のような数字統計の見出しコピー記事は、教室規模・統廃合議論で教員が参照する余地がある。また、文科省統計値の二次引用としての価値もあるため、parser 段階での一律除外は避け、編集者の digest 採否判断に委ねる。数字統計を弾き始めると政策論議系も巻き込むリスクがあり、境界線が引きにくい。

### B. 厳格フィルター(却下)

- POSITIVE_OVERRIDE を持たず、NG パターンに完全一致した場合は無条件に除外
- 却下理由: 「教員向けセミナー」「教職員向け Google for Education 事例校」のような採用継続したい記事を巻き込む

### C. 現状維持(却下)

- kkn の全件採用 + resemom の現行パターンを継続
- 却下理由: ユーザーフィードバックで明示的に問題提起されており、運用品質を下げる

## 決定

A を採用。

### (1) `src/lib/sources/kkn.ts` にフィルター実装

```typescript
const POSITIVE_OVERRIDE = /(教員向け|教職員向け|学校教員)/;

const NG_PATTERNS: readonly RegExp[] = [
  /(アワード|コンクール|コンテスト|コンペ|オリンピック|甲子園|表現の祭典)/,
  /(ワークショップ|公開講座|出前授業|出張授業|キックオフ講座|特別授業)/,
  /(ウェビナー|シンポジウム|フォーラム|サミット|オンラインイベント|勉強会|説明会|セミナー)/,
  /(応募受付|参加申込|希望校.{0,5}募集|生徒募集|受講生募集|エントリー受付|参加.{0,10}募集中|実施校.{0,5}募集)/,
  /(?:を販売|販売(?:を?開始|中)|発売|を発行|キャンペーン)/,
  /(財団|塾|予備校|協会).{0,50}(募集|開催|主催)/,
  /(冷凍食品|お弁当|点鼻液|栄養ドリンク|スピジア)/,
];

export function isExcludedByTitle(title: string): boolean {
  if (POSITIVE_OVERRIDE.test(title)) return false;
  return NG_PATTERNS.some((pattern) => pattern.test(title));
}
```

`fetch()` 内で `if (isExcludedByTitle(title)) continue;` を呼ぶ。

#### 設計上の注記

- 「セミナー」を NG に含めた上で、POSITIVE_OVERRIDE で「教員向けセミナー」を救う構造
- 「販売を?開始」のみだと「を販売」型(マウス整備済パソコン)を捕捉できないため、`(?:を販売|販売(?:を?開始|中))` で広めに
- 「(財団|塾|予備校|協会).{0,15}(募集|開催|主催)」は kkn のタイトル中の修飾文が長いため `.{0,50}` まで許容(「ちゅうでん教育振興財団、学校教育活動への助成支援および教育実践論文を募集中」を捕捉)

#### 対象外(残す方針)

- **数字統計の見出しコピー**(「子供の数は過去最少…」)→ parser pass、編集者判断

## 影響と運用

### 効果

- kkn の約 40% を占めていたノイズ(コンテスト・塾財団告知・食品医薬品)を parser 段階で除外
- resemom の取りこぼし 8 カテゴリ(気象・天文・学園祭・留学フェア・教員業務外資格・Studyplus・商業フェス・イード)を継続的に抑止
- 両 source とも採用継続記事(教員業務支援 / 政策発表 / ICT 事例)は引き続き通過
- 週次 digest 編集の負荷軽減

### 監視

- `npm run check:filter:kkn` 32 ケース pass で回帰防止
- `npm run check:filter:resemom` 41 ケース pass で回帰防止
- 自動収集ログで kkn / resemom の採用件数が極端に減った場合(2 週間連続で 0 件など)、過剰除外を疑い override 拡張を検討

### 厳格 POSITIVE_OVERRIDE のリスク観測項目

- 「中高教員向け◯◯セミナー」など、教員業務支援だが複合表記で block される記事の取りこぼし
- 「校務 DX セミナー」「働き方改革 ウェビナー」など、教員業務の核心語を含むが「教員向け」が明示されないため block される記事
- これらは観測結果に応じて後続 ADR で override 拡張(例: `校務.{0,5}DX` / `働き方改革.{0,5}(セミナー|ウェビナー)` を override に追加)

## 撤回 / 再検討の条件

- kkn / resemom の採用件数が想定を下回り続け、運用上「取りこぼし > ノイズ削減効果」と判断された場合
- 厳格 POSITIVE_OVERRIDE で誤って block される教員向け記事が観測ログで一定数(例: 2 週間で 5 件以上)積み上がった場合 → override 拡張 ADR(0040 以降)で個別救済
- 配信元(kkn / resemom)が編集方針を変更し、ノイズ比率が大きく変動した場合 → 再観測の上で本 ADR を更新

## 遡及適用(2026-05-11)

PR #124(parser 強化)では既存 `src/data/articles/*.json` には触れず、cron 次回以降の新規収集分から本 ADR の NG_PATTERNS が適用される設計だった。本セクションは parser 強化前に取り込まれていたノイズ記事を遡及削除した記録。

### 実行手順

1. `scripts/clean-kkn-resemom-noise.ts` を新規作成(`scripts/clean-mext-off-scope.ts` の `--dry-run` 構造、`scripts/clean-nier-noise.ts` の 2 段階フィルター内訳出力を参考)

4. 本実行で 13 ファイル(2026-04-26 〜 2026-05-10)を書き換え
5. 削除した 46 件全件を `src/data/excluded-article-ids.json` に追加(ADR 0020 機構で RSS 再配信時の予防的 denylist として運用)

### 厳格 POSITIVE_OVERRIDE で意図的に block された kkn 記事

ADR §「厳格 POSITIVE_OVERRIDE のリスク観測項目」に予告した類型が遡及削除でも観測された:

- `教科書AIワカル...割引キャンペーン`(NG: キャンペーン)
- `中高教員・教委向けセミナー(カシオ×八千代エンジニヤリング)`(NG: 複合 override 対象外)
- `スクールタクト GIGA 開きウェビナー`(NG: ウェビナー)
- `ロイロ NEXT GIGA サミット`(NG: サミット)
- `学校施設事例集 vol.3 発行(EDIX 配布)`(NG: 発行)

これらは ADR 採用時の「過剰除外を許容、後続 ADR で個別救済」方針通り。継続観測で 2 週間に 5 件以上の取りこぼしが積み上がれば override 拡張 ADR(0040 以降)で救済する。

# 0019. 対象読者の遡及適用とリセマム NG ワード再設計

- 状態: 採用
- 日付: 2026-05-01
- 関連 ADR: 0007(第 2 層を教育専門紙へ転換)、0018(対象読者を学校教員・教育関係者に統一)
- 関連 PR: TBD(本 ADR の実装)

## 背景

ADR 0018(2026-05-01)で対象読者を「学校教員・教育関係者」に統一した際、「既存記事 JSON に対する遡及的なフィルタ適用 — 行わない(過去ログは不変)」と明記した。しかし採用直後の運用判断で以下が問題になった:

- ADR 0018 採用前に蓄積された記事(2026-04-21〜04-30 の 8 日分)に **保護者向けの記事が 34 件含まれており**、サイトのトップ・アーカイブ・カテゴリ・媒体別ページから可視化される
- 加えて 2026-05-01 朝の cron(PR #66)が ADR 0019 採用前のフィルタで取り込んだ 4 件も同基準で削除対象になる(本 PR でまとめて 38 件削除に拡大)
- 読者像の単線化を文書(README / CLAUDE.md / `/about`)で宣言しても、データ表示が不整合だと宣言が空文化する
- リセマム由来の記事 53 件中 38 件(72%)が「保護者の私的関心に閉じる純情報」(ADR 0018 §6.3)に該当しており、これは将来の収集でも同種記事が継続的に流入することを意味する → 既存の `isExcludedByTitle()` の NG ワード(14 パターン)では捕捉できていない

ユーザー指示(2026-05-01 後半)を踏まえ、ADR 0018 の不変条項を限定的に上書きし、**(a) 既存記事の遡及削除** と **(b) リセマム NG ワード再設計** を 1 PR で実施する。

## 検討した選択肢

### A. 遡及削除 + NG ワード再設計を 1 PR で実施(採用)

メリット:
- 文書(ADR 0018)とデータ(JSON)の整合性を即時に取り戻す
- 既存 34 件の削除は ID 指定で精密に行えるため、誤削除リスクが低い
- NG ワード再設計を併走させることで、将来の同種記事流入を 79%(27/34)程度まで自動抑止できる

デメリット / 緩和策:
- ADR 0018 §「別 PR に分離する変更」「既存記事 JSON に対する遡及的なフィルタ適用 — 行わない」を覆すことになる → 本 ADR で背景と判断根拠を明示する
- 過去の運用記録としては git 履歴(削除コミット)が監査ログとして機能するため、JSON 上から物理削除しても情報損失は最小

### B. Soft delete(`hidden: true` フラグを記事スキーマに追加、UI で非表示化)(却下)

却下理由:
- Article schema に `hidden` フィールドを追加すると、Zod バリデーション・既存コード・将来の D1 移行に影響
- ハードデリートでも git 履歴で監査可能なため、複雑度を上げる必要がない
- 「削除した記事の理由を記録する」ニーズは ADR 本体と PR 本文のカテゴリ表で代替できる

### C. NG ワード再設計のみ(既存記事は不変)(却下)

却下理由: 既存 34 件の不整合が解消されない。ADR 0018 の宣言と現状データのギャップが残る。

## 決定

A を採用。

### 削除対象の判定基準(ADR 0018 §6.3 を運用ルールに具体化)

以下のカテゴリは「保護者の私的関心に閉じる純情報」として削除対象。

| カテゴリ | 例 |

### 実装範囲

#### (1) 既存記事の遡及削除(ハード削除)

9 ファイル(`2026-04-21.json` 〜 `2026-05-01.json`)から 38 件の記事(全て `sourceId: resemom`)を ID 指定で物理削除。

| ファイル | 削除前 | 削除後 | 削除数 |
|---|---:|---:|---:|
| 2026-04-21.json | 4 | 2 | 2 |
| 2026-04-22.json | 16 | 9 | 7 |
| 2026-04-23.json | 14 | 10 | 4 |
| 2026-04-24.json | 23 | 19 | 4 |
| 2026-04-25.json | 7 | 6 | 1 |
| 2026-04-27.json | 35 | 30 | 5 |
| 2026-04-28.json | 21 | 15 | 6 |
| 2026-04-30.json | 24 | 19 | 5 |

非 resemom ソース(mext / chukyo / nier / nikkyo / kkn / kyodo)は今回の対象外(タイトル走査で削除候補ゼロを確認済)。

2026-05-01 分の 4 件は同日朝の cron(PR #66、ADR 0019 採用前のフィルタで稼働)が取り込んだもので、本 PR の rebase 時に同基準で追加削除した(B: 朝日新聞「未来をつくる学びテスト」7/12 小3対象 / A: 立命館大、地域交流イベント / D: 重田教育財団「海外留学奨学金」5名募集 / E: 大学生への仕送り、家賃抜き平均 3.3 万円)。

#### (2) リセマム NG ワード再設計

`src/lib/sources/resemom.ts` の `NG_PATTERNS` を 14 パターンから 26 パターンに拡張:

- A. 親子・商業施設の体験イベント(7 パターン): キッザニア / ディズニー / サンシャイン水族館 / ハリーポッター / スタディツアー / 謎解き / 親子フェア系
- B. 児童生徒個別の参加募集 / コンクール / 講座(2 パターン): 学年語+募集語の組み合わせ、メンバー募集
- C. スクール紹介(3 パターン): 教育フェア / プリスクール / インターナショナルスクール
- D. 個別奨学金 / 個別給付(2 パターン): 学生+奨学金+募集、学年+最大 N 万円+給付
- E. キャリア意識調査 + 家庭ライフスタイル(7 パターン): なりたい職業 / 子供・孫 / 年卒+就職 / 読み聞かせ / テーマパーク / ショッピングモール / 住宅補助 / 熱中症+住居

検証結果:
- **削除対象 34 件中 27 件(79%)** を新 NG_PATTERNS で自動捕捉
- **残す 13 件中 0 件** で false positive(新フィルタによる誤除外)
- 残り 7 件の取りこぼし(汎用イベント語彙で正規化困難)は editor review に委ねる

### 文書整合

- `src/pages/changelog.astro` に 1 行追加(ユーザー価値: 「対象読者外の保護者向け記事を整理」)
- ADR 0018 を遡及的に書き換えはしない(ADR 不変原則)。本 ADR が 0018 §「別 PR に分離する変更」を限定的に上書きする位置付け

## 影響と運用

- 既存記事の総数: 205 → 167(-38、-18.5%)。リセマムの存在感が大幅に低下
- 今後のリセマム由来流入は新 NG_PATTERNS で 79% 程度自動除外。残りは編集者の週次ダイジェスト編集ワークフローで判断
- Pagefind 検索インデックスは次回ビルドで再生成(削除記事は検索結果からも消える)
- 媒体別ページ(`/sources/resemom/`)とアーカイブ(`/archive/`)の表示件数が反映される

## 再検討の条件

- NG_PATTERNS の false positive(教員視点で関連する記事を誤除外)が観測された場合、該当パターンを緩和する
- リセマム由来の流入が極端に減少し編集対象が枯渇する場合、NG_PATTERNS を緩めるか、ReseEd(リセマム教員向け姉妹サイト、ADR 0007 §「再検討の条件」)の追加採用を検討する
- 他ソース(kkn / nikkyo 等)で対象読者外の記事流入が顕在化した場合、本 ADR と同じ枠組みで個別対応する

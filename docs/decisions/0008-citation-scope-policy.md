# 0008. 引用範囲遵守ポリシーと削除依頼窓口

- 状態: 採用
- 日付: 2026-04-25
- 関連 PR: TBD(`feat/sprint2-batch2-major-media`)

## 背景

ADR 0007 で第 2 層を教育専門紙 + 共同通信 RSS に転換した結果、第 2 層 4 媒体のうち 3 媒体(日本教育新聞 / 教育家庭新聞 / 共同通信)は、利用規約に「無断転載禁止」「引用のみ許可」等の文言があり、edu-watch 側がどう運用するかで合法性が分かれる。

日本の著作権法 32 条 1 項は「公表された著作物は、引用して利用することができる」と定めており、判例上は (a) 引用部分が明確に区別されていること、(b) 引用される側が従、引用する側が主であること、(c) 引用の必然性、(d) 出典の明示、を満たせば適法とされている。

edu-watch がこの「引用」の枠組みに収まる運用を続けるためには、技術的・運用的な歯止めを ADR レベルで明文化しておく必要がある。

## 検討した選択肢

- **A) 各媒体ごとに個別の運用ルールを定める**(運用が複雑になり抜け漏れが起きる)
- **B) 全媒体に共通する 5 要件を ADR で固定し、新媒体追加時もこの要件を満たすかで判断する**

## 決定

**B) 全媒体共通の 5 要件 + 削除依頼窓口の運用** を採用。

### 引用範囲遵守 5 要件

第 2 層(およびそれに準じる引用元)の全媒体について、以下を必須とする:

1. **出典の明示** — 各記事カードに `出典: <媒体名>` を表示。媒体名は媒体側の正式名称を使う(例: 「リセマム」「日本教育新聞」「教育家庭新聞」「共同通信」)
2. **公式 URL への直リンク** — 記事タイトルのクリックは元媒体の公式 URL に遷移させる。フレーム内表示、プロキシ経由表示、URL ラッパー(短縮 URL / リダイレクト URL)を経由させない
3. **要約は RSS 配信のものをそのまま使う** — 独自に長くしない、要約しない、AI で書き換えない。RSS の `<description>` を改変せずに表示する。本文取得・本文要約は行わない
4. **タイトルの改変禁止** — タイトルを編集者が短縮・意訳・煽情化しない。媒体側のタイトルをそのまま掲載する(PRD §6 と整合)
5. **削除依頼窓口の明示** — `/about` ページに「掲載削除のご依頼」セクションを設け、`takedown@edu-evidence.org`(Cloudflare Email Routing で運営者へ転送)宛に申請できる旨を明記。**申請受領後 24 時間以内に該当記事を削除する**ことを宣言する

### 取得項目の限定

RSS から取得して edu-watch 上に表示する項目は、以下に限定する:

- 記事タイトル(原文ママ)
- 公式 URL(直リンク)
- 公開日時
- 媒体名
- RSS 配信された短い要約(`<description>`、改変禁止)
- カテゴリ(媒体側が `<category>` で配信している場合のみ)

これ以外(本文、画像、著者名、コメント等)は取得・表示しない。

### `/about` の構成

`/about` ページに以下のセクションを追加する(Sprint 3 のフロント実装時に対応):

- 「収集ポリシー」: ADR 0007 で定めた採用ソース一覧と取得方式
- 「引用の立場」: 5 要件と著作権法 32 条への準拠について
- 「掲載削除のご依頼」: 削除依頼窓口メール、申請後の対応フロー、24 時間以内対応の宣言

### 媒体側からの問い合わせ対応

将来、第 2 層のいずれかの媒体から運用方針の照会・是正依頼があった場合は、**まず該当媒体の記事掲載を一時停止**し、本人と協議のうえで運用方針を見直す。協議の結果は新規 ADR にする。

## 帰結

### 良い帰結

- 著作権法 32 条「引用」の要件を技術的に担保できる
- 媒体側からの是正依頼に即応できる窓口が用意される
- 新ソース追加時に同じ判断軸が使える(チェックリスト化が容易)
- edu-evidence 本家との編集ポリシーの一貫性が保たれる(タイトル改変なし、編集者意見なし)

### トレードオフ

- 「edu-watch 独自の見出し付けで読者の関心を引く」という運用ができない(原文ママ縛り)
- 要約の質が媒体側の RSS の質に依存する(媒体によっては要約が短すぎる / 長すぎる場合あり)
- 削除依頼への 24 時間以内対応は運営者の負荷になる(休暇中の不在対応をどうするかは別途検討)

## 撤回 / 再検討の条件

- 媒体側から「全文転載 OK」「タイトル編集 OK」等の正式許諾が下りた媒体が現れた場合(該当媒体のみ別運用とする)
- 削除依頼が想定を大きく超える頻度で来た場合(運用負荷が破綻する閾値を超えた時点で見直し)
- 著作権法 32 条の解釈を変える判例 / 法改正が出た場合
- `takedown@edu-evidence.org` のメール転送先・対応体制を変更する場合

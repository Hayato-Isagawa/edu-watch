# 0001. 双葉(cotyledon)非対称ロゴで芽吹きを表現

- 状態: 採用
- 日付: 2026-04-24

## 背景

EduWatch JP は EduEvidence JP の姉妹サイトとして立ち上がる。植物モチーフの 4 サイトブランド体系(edu-evidence ADR 0001)で双葉(cotyledon)を担うサイトと位置づけたが、具体的なシンボル形状の決定が必要だった。

「成熟した葉(edu-evidence)」と並べたときに、(a) 同じ植物体系であることが伝わり、(b) 編集スタンスの違い(ストック / フロー、長期 / 時事、確立 / 萌芽)が形からも読めるデザインが望ましい。

## 検討した選択肢

- **A) 完全に左右対称な双葉**(edu-evidence の葉と並列で、シンプルな対比)
- **B) 左右非対称な双葉**(片方の葉が小さい / 片方が先行する。茎にわずかな傾き)
- **C) 双葉ではなく芽(sprout、茎+1 枚の小さい葉)モチーフ**

## 決定

**B) 左右非対称な双葉** を採用。

- 左の葉を小さく、右の葉が一歩先行する形
- 茎にわずかな傾きを入れて静止画でも動きを感じさせる
- アクセント色は **ディープネイビー `#1e4a6e`**(edu-evidence の深緑と対比、夜のニュース感)
- 実装: `src/components/Logo.astro` の inline SVG + `currentColor` 継承

## 帰結

### 良い帰結

- 「育っている途中」「動きがある」感覚が静止画でも伝わる
- 成熟した葉(edu-evidence)との視覚的対比が明確
- `currentColor` 継承により、テーマ色の差し替えが SVG を触らずに完結

### トレードオフ

- 完全対称ロゴと比べてレイアウト時に重心が偏るため、配置時に調整が必要な場面がある
- favicon サイズで非対称さがつぶれて見える可能性がある(現状は許容)

## 撤回 / 再検討の条件

- ブランド体系全体の見直しが入った場合
- favicon / アプリアイコンサイズで非対称さが破綻していると判明した場合

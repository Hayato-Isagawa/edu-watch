# 0051. リセマム include 方式 教員視点キーワードフィルター(NG_PATTERNS との AND ゲート)

- 状態: 採用
- 日付: 2026-05-23
- 関連 ADR: 0007(Tier 2 教育専門紙への転換)/ 0018(対象読者を学校教員・教育関係者に統一)/ 0019(リセマム NG ワード再設計)/ 0020(narrow パターン)/ 0024(取りこぼし予防)/ 0039(8 カテゴリ追加)
- 関連 PR: TBD(本 ADR の実装)
- 撤回 / 再検討トリガー: 本 ADR §「撤回 / 再検討の条件」参照

## 背景

ADR 0019 / 0020 / 0024 / 0039 で resemom の `NG_PATTERNS`(除外型)を段階的に育成し、現時点で 30+ パターンを保有している。しかし NG-only 方式は次の構造的問題を抱える:

1. **新カテゴリのノイズが恒常的に出現する**。受験産業色 → 親子体験イベント → 気象 → 商業フェスと、ユーザー方針(ADR 0018)から外れる軸が次々に現れ、後追い拡張が継続して必要になる
2. **取りこぼし観測のコストが高い**。除外漏れは毎回ユーザーまたは collector ログを目視確認しないと発見できず、ADR 0019(11 カテゴリ)→ 0020(narrow 6 件)→ 0024(narrow 4 件)→ 0039(8 カテゴリ)と 4 度の追加 ADR を要した
3. **誤除外の検出も同様にコストが高い**。各 NG パターン拡張のたびに `scripts/check-resemom-filter.ts` に手動でケースを追記する必要がある

2026-05-15 〜 2026-05-21 の 7 日間で resemom から収集された 36 件を観測した結果、NG 除外後の残り 33 件(NG_PATTERNS 通過分)の中に、教員視点で価値の低い記事が広範に含まれていた。具体的には商業 PR(LINE 読書アプリ・モンチッチ絵文字)、自治体施策(水道料金無償化・麻しんワクチン)、レジャー施設(淡路ファームパーク)、大学広報(北大公開・国立科博展)など、NG パターンの追加だけでは捕捉しきれない雑多なジャンルが混入していた。

NG-only を継続拡張するより、**「教員視点キーワードに該当しなければ採用しない」という include 方式に切り替える方が、運用コストとノイズ削減効果のバランスが良い**段階に達したと判断した。kkn(教育家庭新聞)では ADR 0039 で既に類似方針(`POSITIVE_OVERRIDE`)を採用済みだが、kkn は POSITIVE_OVERRIDE が AND ではなく NG への上書きとして働いていた。resemom では NG_PATTERNS と include 方式の AND ゲートで二重に絞り込む方式を採用する。

### kkn との運用方針差

kkn は GIGA 関連の網羅性担保が優先で、ADR 0039 の `POSITIVE_OVERRIDE` は「教員向け」「教職員向け」「学校教員」の 3 語に絞った narrow override として働く。一方 resemom は保護者向け媒体で対象読者の乖離がもともと大きく、include キーワードを 46 語に拡張して教員視点のシグナルを広く拾う設計とした。両 source の媒体特性に応じた差別化である。

## 検討した選択肢

### A. NG_PATTERNS の継続拡張(却下)

- 現状の NG-only 方式を維持し、新カテゴリのノイズが出るたびに ADR で追加
- 却下理由: 4 度の追加 ADR で 30+ パターンに膨らみ、なお新カテゴリが継続して出現する。後追い対応の構造から脱却できない

### B. include 方式 + NG_PATTERNS の AND ゲート(採用)

- `EDUCATION_PATTERNS`(46 語、9 群)を新設し、`isEducationallyRelevant(title, summary?)` 関数を export
- `fetch()` で `!isExcludedByTitle(title) && isEducationallyRelevant(title, summary)` の AND を要求
- NG_PATTERNS 通過後の残りから、教員視点キーワードを含まない記事をさらに除外する二重ゲート
- 過剰除外リスクは過去 1 週間データで実地検証(後述)

### C. include 方式単独 / NG_PATTERNS の廃止(却下)

- include 方式に置き換えて NG_PATTERNS を削除
- 却下理由: 「中学受験」「偏差値」「ランキング」のような受験産業色は教員視点キーワード(「教育」「学校」など)を同時に含むことが多く、include 単独では弾けない。NG_PATTERNS との AND は補完関係にあり、両方残すのが堅牢

## 決定

B を採用。具体内容は以下のとおり。

### (1) `EDUCATION_PATTERNS`(46 語、9 群)

`src/lib/sources/resemom.ts` に以下を定義する。

| 群 | キーワード |
|---|---|
| 学校制度・職位 | 教育 / 学校 / 教員 / 教師 / 教職 / 教科書 / 教科 / 校長 / 教頭 / 学級 |
| 児童生徒・学習 | 児童 / 生徒 / 学習 / 学力 / 指導要領 / 部活動 / 部活 |
| 生徒指導課題 | いじめ / 不登校 / 特別支援 / 発達障害 |
| 教員人事・労働 | 給特法 / 働き方改革 / 教員研修 / 教員養成 / 教員採用 |
| ICT 教育 | GIGA / ICT / タブレット / プログラミング教育 / デジタル教科書 |
| 教育行政 | 中教審 / 文科省 / 文部科学 / 教育委員会 / 通知 / 答申 |
| 進路 | 入試 / 進路 |
| 教育機会・健康・安全 | フリースクール / 教育機会確保法 / カウンセラー / SC / SSW / 校則 / 学校給食 / 保健室 |
| 家庭・経済 | 家庭学習 / 家庭との連携 / 保護者会 / 経済格差 / 子どもの貧困 / 子供の貧困 |
| カリキュラム | 教科担任制 / 探究学習 / 探究的な / 学習評価 / カリキュラム |

意図的に除外した語: 「家庭」「子ども」「保護者」「親」(単独)。これらはノイズリスクが高く、商業 PR の見出し常套句として頻出する。

### (2) `isEducationallyRelevant(title, summary?)`

- `title` または `summary` のいずれかに `EDUCATION_PATTERNS` のいずれかが一致すれば `true` を返す
- summary 未指定時は title 単独で評価
- 純粋関数として export し、テスト容易性を担保

### (3) `fetch()` への統合

```ts
if (isExcludedByTitle(title)) continue;
const summary = item.contentSnippet?.trim() || item.content?.trim() || undefined;
if (!isEducationallyRelevant(title, summary)) continue;
```

NG_PATTERNS で先に弾き、include 方式で更に絞り込む。AND ゲート。

### (4) NG_PATTERNS の追加(本 ADR で 5 件)

過去 1 週間データの観測で見つかった取りこぼしを include 方式に頼らず NG 側で明示的に弾く。

| パターン | 対象タイトル例 | 理由 |
|---|---|---|
| `/(?:大学\|高校)野球.{0,10}(?:リーグ\|大会\|選手権)/` | 國學院大が7季ぶりV…東都大学野球春季リーグ / 【高校野球】北海道春季大会 | スポーツ大会試合結果 |
| `/六大学野球/` | 東京六大学野球、勝ち点3明大のスタメン…進学校・強豪校 | 「進学校」内の「学校」substring が `EDUCATION_PATTERNS` を誤通過するため NG 側で先行除外 |
| `/(?:子ども\|こども).{0,3}フェスタ/` | 手づくり遊園地が登場「こどもフェスタ」 | 子ども向け一般イベント |
| `/ジュニア大使/` | ブルネイでホームステイと交流、小5-高2のジュニア大使募集 | 国際交流プログラム参加募集 |
| `/ホームステイ.{0,10}(?:派遣\|募集)/` | 同上、上記の裏フォールバック | ジュニア大使語が将来変わっても捕捉する narrow ガード |

## 検証

### 過去 1 週間データへの適用(2026-05-15 〜 2026-05-21)

`scripts/check-resemom-filter-past-week.ts` で `src/data/articles/2026-05-1[5-9].json` + `2026-05-2[0-1].json` の resemom 記事 36 件に新フィルタを適用した結果:

| 区分 | 件数 | 割合 |
|---|---|---|
| 通過 | 12 | 33.3% |
| NG 除外 | 5 | 13.9% |
| 非教育キーワード除外 | 19 | 52.8% |
| 合計除外 | 24 | 66.7% |

### NG 除外 5 件(妥当性確認済み)

- 東京六大学野球、勝ち点3明大のスタメン…進学校・強豪校など多彩な出身校 → `/六大学野球/`
- 國學院大が7季ぶりV、青学の7連覇阻止…東都大学野球春季リーグ → `/(?:大学|高校)野球.{0,10}(?:リーグ|大会|選手権)/`
- 手づくり遊園地が登場「こどもフェスタ」千葉・市原市 → `/(?:子ども|こども).{0,3}フェスタ/`
- ブルネイでホームステイと交流、小5-高2のジュニア大使募集 → `/ジュニア大使/`
- 【高校野球】北海道春季大会、16校の組合せ決定 → `/(?:大学|高校)野球.{0,10}(?:リーグ|大会|選手権)/`

### 通過 12 件(教員視点での妥当性)

通過した記事には「校長」「教員」「探究学習」「教科」「入試」「学習」など `EDUCATION_PATTERNS` の語が title または summary に一致しており、教員視点で関心がある可能性のある内容に絞られている(例: 通信制高校探究学習、教員同士の結婚 Teacher's Shift、大学入学共通テスト 2027 まとめ、慶應 3 年編入学試験新設、ヨンデミー SAPIX 導入、東大理科三類学習法)。一部に保護者向けエンタメ寄り(ケンブリッジ大学 800 年教育、9 歳の壁を突破する読書)が含まれるが、これは include 方式の許容範囲とする(編集者の digest 採否判断に委ねる)。

### 意図的に採用しない判断(STEM オリンピック等)

- 科学のオリンピック・ISEFで日本代表が世界1位含む8件受賞(2026-05-18 resemom)
- ヨーロッパ女子情報オリンピック、4人全員メダル受賞…洛南高生が金(2026-05-19 resemom)

これらは個別生徒の受賞報告であり、教員視点での政策・実務インパクトが低いため、`EDUCATION_PATTERNS` 不一致による自動除外を維持する。STEM オリンピック語(`(?:科学|数学|情報|物理|生物)オリンピック`)を `EDUCATION_PATTERNS` に追加して採用するか否かは、撤回トリガーに到達した時点で再検討する。

### テストスクリプト

`scripts/check-resemom-filter.ts` に 5 件のケース(NG 5 件: 國學院大東都大学野球 / こどもフェスタ / ジュニア大使 / 東京六大学野球 / 高校野球北海道春季大会)を追加した。テスト結果: **46 passed / 0 failed**。

`npm run check`: 0 errors / 0 warnings(25 hints は既存 zod deprecation のみ、本 ADR 範囲外)。

## 撤回 / 再検討の条件

以下のいずれかが観測された場合、本 ADR を再評価する。

1. **過剰除外**: 教員視点で重要な政策・現場情報(中教審答申速報、文科省通知、給特法改正、現場の実践事例など)が `EDUCATION_PATTERNS` 不一致で漏れる事案が 1 件でも発生
2. **過剰通過**: NG_PATTERNS と include 方式の AND を通過した記事のうち、digest 採否で 3 セッション連続で全件不採用が続く(include 方式の signal-to-noise 比が悪化している兆候)
3. **媒体方針変更**: resemom が大規模リニューアルや編集方針転換を行い、本 ADR の前提(保護者向け媒体)が崩れる
4. **kkn との運用一貫性**: kkn 側で resemom と同型の include 方式に切り替える判断が出た場合、両 source の関数シグネチャ・キーワード分類を統一する形で本 ADR を更新

## 関連 ADR との関係

- **ADR 0007**: Tier 2 を教育専門紙へ転換した方針を継承。resemom は Tier 2 として残すが、include 方式で対象読者方針(ADR 0018)との整合を強化
- **ADR 0018**: 対象読者「学校教員・教育関係者」を運用層で具体化する手段。本 ADR の include 方式は ADR 0018 の implementation
- **ADR 0019 / 0020 / 0024 / 0039**: NG_PATTERNS の段階的拡張を継承。本 ADR で AND ゲートの片側として継続運用
- **ADR 0039 §kkn**: kkn の `POSITIVE_OVERRIDE` は narrow override(NG への上書き)、本 ADR の resemom include は AND ゲート。両 source の媒体特性に応じた差別化として共存

# 0053. EDUCATION_PATTERNS 拡充: 情報モラル / ネットパトロール / キャリア教育

- 状態: 採用
- 日付: 2026-05-24
- 関連 ADR: 0007(Tier 2 教育専門紙への転換)/ 0018(対象読者を学校教員・教育関係者に統一)/ 0019(リセマム NG ワード再設計)/ 0051(EDUCATION_PATTERNS include 方式の導入)
- 関連 PR: #192(本 ADR の起点となる個別維持)/ TBD(本 ADR の実装)
- 撤回 / 再検討トリガー: 本 ADR §「撤回 / 再検討の条件」参照

## 背景

ADR 0051 で `EDUCATION_PATTERNS`(9 群 46 語)による include 方式を採用し、`isExcludedByTitle()`(NG 除外型)との AND ゲートで resemom の絞り込みを運用してきた。同 ADR §「撤回 / 再検討の条件」§1 は次の発火条件を定めている:

> 過剰除外: 教員視点で重要な政策・現場情報(中教審答申速報、文科省通知、給特法改正、現場の実践事例など)が `EDUCATION_PATTERNS` 不一致で漏れる事案が 1 件でも発生

PR #192(2026-05-22 起票、ADR 0051 の resemom フィルタを過去データへ後追い適用する backfill)の検証時点で、この §1 が以下の 2 件で発火した:

1. `resemom-2026-04-27-29914904e409c0b9` 「公務員志望、親の職業や安定志向が影響…4 大学が 2,643 人調査」 — 立命館大学・関西学院大学・筑波大学・大阪経済大学による 2,643 人の進路選択要因調査。進路選択要因系の研究調査記事だが、ADR 0051 既存パターンでは title/summary とも非マッチ
2. `resemom-2026-05-12-e49ec95f8a52cd91` 「中高生ネットパトロール、特に問題ある書込み増加…千葉県」 — 千葉県の 2025 年度青少年ネット被害防止対策事業(ネットパトロール)。自治体の情報モラル指導施策だが、ADR 0051 既存パターンでは title/summary とも非マッチ

両記事は PR #192 で個別維持(denylist 例外)としたが、ADR 0051 §1 トリガーの本来の対応方針は EDUCATION_PATTERNS の拡張である。個別維持を続けると denylist 管理コストが膨らみ、将来同種記事が出るたびに人手判断が必要になる。

## 検討した選択肢

### A. 個別維持パターンの継続(却下)

- 今回の 2 件と同様、今後も該当記事は denylist 例外で個別維持
- 却下理由: ADR 0051 §1 トリガーの発火は本来 `EDUCATION_PATTERNS` の拡張で対応すべき設計。個別維持を増やすと denylist 管理コストが膨らみ、PR #192 のような後追い backfill の手間が継続する

### B. EDUCATION_PATTERNS に 3 語追加(採用)

- ICT 教育群に `/情報モラル/` `/ネットパトロール/` を追加(2 語)
- 進路群に `/キャリア教育/` を追加(1 語)
- 新規 3 語、群分類は 9 群維持、語数 46 → 49

### C. より広い拡張 / 群再編(却下)

- 「進路選択要因」「研究調査」「自治体施策」「学生意識調査」などの上位概念で大幅追加
- 却下理由: 範囲が広すぎてノイズリスク上昇。ADR 0051 で意図的に除外した「家庭」「子ども」「保護者」「親」(単独)と同じ罠を踏みやすい。今回観測された具体的キーワードに絞った narrow 拡張が、ADR 0051 §1 トリガー再発防止と過剰 retain 抑制のバランスとして妥当

## 決定

B を採用。具体内容は以下のとおり。

### (1) ICT 教育群の拡張(+ 2 語)

既存 `/GIGA/ /ICT/ /タブレット/ /プログラミング教育/ /デジタル教科書/` に `/情報モラル/ /ネットパトロール/` を追加する。

| 新パターン | 対象タイトル例 | 効果 |
|---|---|---|
| `/情報モラル/` | 保護者の情報モラル、家庭での声かけ | 既存パターン非マッチの記事を新規 retain。学習指導要領で「情報モラル教育」が明示されており、教員視点の重要施策キーワード |
| `/ネットパトロール/` | 中高生ネットパトロール、特に問題ある書込み増加…千葉県 | PR #192 維持記事を自動 retain 化。自治体・教育委員会が実施する青少年ネット被害防止施策の典型語 |

### (2) 進路群の拡張(+ 1 語)

既存 `/入試/ /進路/` に `/キャリア教育/` を追加する。

| 新パターン | 単独効果 | 登録の意義 |
|---|---|---|
| `/キャリア教育/` | 既存 `/教育/` で常に retain されるため単独効果は冗長(「キャリア教育」を含む title/summary は必ず「教育」も含む) | 群分類の明示性 + 将来 `/教育/` を狭める判断が出た場合の保険として明示登録 |

### (3) NG_PATTERNS への追加: なし

本 ADR では include 側の拡張のみを行う。今回観測された取りこぼし 2 件は include 不一致が原因であり、NG_PATTERNS は引き続き ADR 0019/0020/0024/0039/0051 の蓄積をそのまま運用する。

## 撤回 / 再検討の条件

以下のいずれかが観測された場合、本 ADR を再評価する。

1. **進路選択要因系の再発**: 「公務員志望」「進路選択要因調査」「将来希望職業調査」など PR #192 維持 1 と同種の記事が新たに 2 件以上個別維持された場合、`/キャリア教育/` よりさらに narrow なキーワード(`/進路選択/` `/職業意識/` `/将来希望/` など)の追加を検討
2. **ICT 教育群の再 NG 化**: `/情報モラル/` `/ネットパトロール/` の追加により、情報モラル教育を装ったスクール PR や非教員向け IT セキュリティ記事が retain されるノイズが発生した場合、NG_PATTERNS への narrow 追加(`/プリスクール.{0,10}情報モラル/` 等)を検討
3. **digest 引用済み 3 件への対応**: 本検証で発見した digest 引用済み削減対象 3 件(親の関与調査 / AI 利用調査 / 数字統計コピー)系の取りこぼしが今後も継続する場合、ADR 0051 §1 トリガー再発として別 ADR で再評価
4. **kkn との運用一貫性**: kkn 側で resemom と同型の include 方式に切り替える判断が出た場合、両 source の関数シグネチャ・キーワード分類を統一する形で本 ADR を更新

## 関連 ADR との関係

- **ADR 0051**: 本 ADR は ADR 0051 §「撤回 / 再検討の条件」§1 発火を根拠とする直接的な後続。EDUCATION_PATTERNS の継続的育成の一環として、ADR 0051 を撤回せず拡張する形式を採用
- **ADR 0018**: 対象読者「学校教員・教育関係者」の方針を強化する。情報モラル指導 / ネットパトロール / キャリア教育 はいずれも学習指導要領または自治体・教育委員会施策で扱われ、教員視点で重要な施策・概念
- **ADR 0019 / 0020 / 0024 / 0039**: NG_PATTERNS の段階的育成方針を継承。本 ADR は include 側の拡張のみで、NG_PATTERNS は据え置き
- **ADR 0007**: Tier 2 教育専門紙への転換方針を継承。resemom を Tier 2 として運用しつつ、include 方式の継続的拡張で対象読者方針(ADR 0018)との整合を強化
